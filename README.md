<h2>Getting Started</h2>
  <p>
      To get started with the Fluke Social Network API, you will need to have
      Node.js v12 or later and npm or yarn installed on your machine. Then, run
      the following commands to install the project dependencies and start the
      application in development mode:
    </p>

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
